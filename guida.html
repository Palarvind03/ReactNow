React attraverso la libreria

Il mettodo più facile e veloce per poter iniziare ad utilizzare React è attraverso importazione della sua libreria, con i cosidetti CDN.

Questo processo non richiede alcuni requisiti di installazione da soddisfare o strumenti complicati da utilizzare, dunque ti servono solamente una connessione a internet e pochissimi secondi del tuo tempo.

L'operazione consiste, semplicemente, nel riportare nel tuo codice HTML i seguenti script:

<script crossorigin src="https://unpkg.com/react@17/umd/react.development.js"></script>
<script crossorigin src="https://unpkg.com/react-dom@17/umd/react-dom.development.js"></script>


Attenzione, però, quei due script devono essere messi alla fine di tutti gli elementi DOM della pagina HTML(es. div, p, a...).
Per cui è consigliato metterili nel body della tua pagina, però alla fine tutti i tag...



Infine, se non ti interessa includere tutte le funzionalità, ma soltanto quelle principali...potresti povare ad usare la versione minificata: 

<script crossorigin src="https://unpkg.com/react@17/umd/react.production.min.js"></script>
<script crossorigin src="https://unpkg.com/react-dom@17/umd/react-dom.production.min.js"></script>

React attraverso CRA

Create React app (CRA) è un progetto prototipo che nasce con l'obiettivo di rimuovere le difficoltà di configurazione di webpack. Usa, infatti, webpack e fornisce una configurazione già pronta per essere usata in pochi click. In questo modo ci si può concentrare piu sullo sviluppo dell'applicazione senza pensare troppo alle diverse configurazioni.

Requesiti neccessari per l'installazione:

  • Node.js (su windows, Node nel suo set up include anche Npm)
  • Npm


Una volta soddisfatti i requisiti, spostati in una cartella a piacere ed aprire il terimiale(cmd) nella directory in questione. Ora esguire il seguente commando per dare via all'installazione: 

D:\mieiProgetti> npx create-reat-app mia-app

A questo punto, dopo l'installazione, nella tua directory verrà scaricato una caretella(nel mio caso 'mia-app'). Per verificare se tutto sia andato a buon fine; ti basta spostare nella cartella scaricata ed eseguire:

D:\mieiProgetti\mia-app> npm start

Subito dopo l'esecuzione del commando, verrà aperto automaticamente una scheda nel browser e se dopo il caricamento...vedi come rafigurato nel l'immagine sotto; allora complimenti: 'Hai completato l'operazione con successo!'.

Struttura del CRA

La cartella node_module, contiene una serie di librerie necessarie a far funzionare l'ambiente di test oltre che quelle specifiche per il funzionamento di un’applicazione React.


Poi troviamo una cartella public che è paragonabile alla classica cartella principale di un sito Web, in cui è presente un documento index.html, un'icona e un file manifest.json, necessario per lo sviluppo di Progressive Web App.


Infine vi è la cartella src, che contiene una serie di file CSS e JavaScript, e sulla quale andremo a lavorare per creare i diversi file JavaScript che costituiranno la nostra applicazione.

Che cosa è JSX?

JSX sta per JavaScript XML. Utilizzare JSX in react Non è necessario, ma utilizzandolo rende più facile scrivere applicazioni React.



A cosa serve JSX?

JSX ci permette di scrivere elementi HTML in JavaScript e inserirli nel DOM senza usare il metodo createElement() o appendChild().



Con e senza JSX

con:

const guidaReact = <h1>React Now!</h1>;
ReactDOM.render(guidaReact, document.getElementById('root'));

senza:

const guidaReact = React.createElement('h1', {}, 'React Now!');
ReactDOM.render(guidaReact, document.getElementById('root'));

Come puoi vedere nel primo esempio, JSX ci permette di scrivere HTML direttamente all'interno del codice JavaScript.



Espressioni in JSX

Con JSX puoi scrivere espressioni all'interno di parentesi graffe { }.
L'espressione può essere una variabile o una proprietà React o qualsiasi altra espressione JavaScript valida. JSX eseguirà l'espressione e restituirà il risultato: 


lib = "React";
const mioTesto =  <h1>{lib+' Now'} è la migliore guida {lib}!</h1>;
              
React Now è la migliore guida React!

Mettodo map()

Il metodo Array.map() consente nel iterare un array e modificare i suoi elementi utilizzando una funzione di callback.
La funzione di callback verrà quindi eseguita su ciascuno degli elementi dell'array.

Ad esempio, supponiamo di avere il seguente vettore:

let vet = [3, 4, 5, 6];


Ora immagina di dover moltiplicare ciascuno degli elementi dell'array per 3. Potresti considerare di utilizzare un ciclo for come segue:

let vet = [3, 4, 5, 6];

for (let i = 0; i < vet.length; i++){
  vet[i] = vet[i] * 3;
}
                
console.log(vet); //output --> [9, 12, 15, 18]



Puoi però utilizzare anche il metodo Array.map() per ottenere lo stesso risultato. Ecco un esempio:

let vet = [3, 4, 5, 6];

let vetModificato = vet.map(function(elemento){
    return elemento *3;
});

console.log(vetModificato); //output --> [9, 12, 15, 18]



La Destrutturazione

La destrutturazione consente di assegnare le proprietà di un array o di un oggetto utilizzando una sintassi simile ai valori letterali di array o oggetti. Questa sintassi inoltre permette di gestire meglio il codice rispetto all'accesso tradizionale alla proprietà.

Senza destrutturare l'assegnazione, potresti accedere ai primi tre elementi di un array in questo modo: 

let vet = [1,2,3]

let primo = vet[0];
let secondo = vet[1];
let terzo = vet[2];

console.log(secondo) //output --> 2


Con la destrutturazione, invece, il codice equivalente diventa più conciso e leggibile 

let vet = [1,2,3]

let [primo, secondo, terzo] = vet;

console.log(secondo) //output --> 2


Altri usi


1) Puoi saltare gli elementi dell'array che vengono destrutturati:

let [,,third] = ["pippo", "pluto", "topolino"];

console.log(first); //output --> "pippo"


2) Puoi catturare tutti gli elementi finali in un array:

let [primo, ...altri] = [1, 2, 3, 4];

console.log(primo); //output --> 1
console.log(altri); //output --> [2, 3, 4]


Destrutturazione degli oggetti

let { matteo, alessio } = { cognome1: "rossi", cognome2: "bruno" };
console.log(matteo); // output --> rossi
console.log(alessio); // output --> bruno


useState Hook

useState è un Hook che ti permette di avere variabili di stato. Basta passare a questa funzione lo stato iniziale e ti restituisce una variabile con il valore dello stato corrente (non necessariamente lo stato iniziale) e un'altra funzione per aggiornare questo valore.


Prima di utilizzare questa funzione però occorre importarlo: 

import React, { useState } from 'react';


sintassi

const [variabile, setVariabile] = useState(1);


Come accedere alla variabile dello useState?

const [msg, setMsg] = useState("ciao");

<p>{msg}</p>


Come modificare la variabile dello useState?

const [msg, setMsg] = useState("ciao");

setMsg("ciao ciao")
<p>{msg}</p>

Le promise

Cos'è la promise(promessa)?

La promise rappresentano il risultato di un'operazione asincrona. Per farla breve puoi pensarlo come un segnaposto per un risultato che alla fine riceverai.


Tre stati di promessa


In attesa → Prima che il risultato sia pronto, una promessa è in uno stato in sospeso.br
Eseguito → Se il risultato è disponibile, diventa soddisfatto.
Rifiutato → Se si verifica un errore, la promessa viene rifiutata.


Sintassi

let miaPromise=new Promise(function(resolve, reject){

  //operazione asincron va qua...

})


mettodo then()

Il mettodo then() viene utilizzato per fare tutte operazioni che verrano eseguire dopo che la promessa avrà dato una risposta. Questo mettodo ha argomenti opzionali:
1) callback per il successo
2) callback per il fallimento


Esempio completo

let miaPromise=new Promise(function(resolve, reject){

  //operazione asincron va qua...

})

miaPromise.then((risposta) => {
  //istruzioni dopo il successo
},(errore) => {
  //istruzioni dopo l'errore
});


Fetch API

L'API Fetch è un'interfaccia moderna che consente di effettuare richieste HTTP ai server dai web browser. Utilizza, inoltre, Promise per fornire funzionalità più flessibili...


Il fetch() richiede un solo parametro che è l'URL della risorsa che vuoi recuperare:

let risposta = fetch(url);


Cosa restuisce?

Il metodo fetch () restituisce una promessa in modo da poter utilizzare i metodi then() e catch() per gestirla:
then() → per eserguire le istruzioni dopo il completamento della richiesta
catch() → per eserguire le istruzioni dopo il fallimento della richiesta


Esempio completo

url = "http://reactNow.it""
  
fetch(url)
   .then(risposta => {
      console.log(risposta);
   })
   .catch(error => console.log("Si è verificato un errore!"))
